What is programming?

Programming is both a highly technical discipline and a medium for artistic expression. Although programming perfectly blends both science and art, it is not without cost. If you are beginning a journey in computer science, prepare to have your patience, tenacity, and sanity tested..

One of the most common questions beginners ask is,
"What programming language should I learn first?"

Are you ready to see for yourself?

You should learn CSS and Design!

CSS is a style sheet language used for describing the presentation of markdown languages like HTML. CSS and Design go hand in hand together, and will be present in any web-based endeavor. If you focus on CSS and Design, you will be able to use those skill in any web development job.

You should learn Ruby!

Ruby is a dynamic, object-oriented programming language designed in Japan. It is commonly used by start-up companies.

You should learn C#!

C# is an object-oriented programming language developed by Microsoft, commonly used with Microsoft's .NET framework. Many corporate and government level jobs use C#.
